# Limit first Matches to standard-single

首个真人 Match 切片只支持现有 `standard-single`：两名 Trainer、每方一个上场位，不接受调用方提供其他 `formatCode` 或自定义规则。Trainer Team 可以包含一至六名成员，Challenge 公开发起方人数并要求接受方选择相同人数的 Team，双方 Snapshot 全员直接进入 Session Roster，不增加二次选人阶段；该格式当前没有种类唯一或道具唯一条款，Match 不额外添加这些竞技限制，未来需要时应新增显式 `official-single` 格式。
