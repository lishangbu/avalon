# 资料依据

本技能主要基于 Kotlin 官方文档，而不是第三方解读。

核心来源：

- Library authors' guidelines introduction
    - https://kotlinlang.org/docs/api-guidelines-introduction.html
- Simplicity
    - https://kotlinlang.org/docs/api-guidelines-simplicity.html
- Readability
    - https://kotlinlang.org/docs/api-guidelines-readability.html
- Consistency
    - https://kotlinlang.org/docs/api-guidelines-consistency.html
- Predictability
    - https://kotlinlang.org/docs/api-guidelines-predictability.html
- Coding conventions
    - https://kotlinlang.org/docs/coding-conventions.html
    - 尤其关注其中的 `Source file names` 与 `Source file organization`
- KDoc
    - https://kotlinlang.org/docs/kotlin-doc.html
- Calling Kotlin from Java
    - https://kotlinlang.org/docs/java-to-kotlin-interop.html
- Explicit API mode API reference
    - https://kotlinlang.org/api/kotlin-gradle-plugin/kotlin-gradle-plugin-api/org.jetbrains.kotlin.gradle.dsl/-kotlin-top-level-extension-config/explicit-api.html

如果你使用中文镜像，可以把 Kotlin 中文站作为辅助阅读入口；但当镜像版本滞后或页面不可用时，应以 Kotlin 官方文档为准。
