# Kotlin 官方来源

本技能依据 Kotlin 官方文档整理，只保留可执行实践，不复制官方正文。

## 编码风格

- Coding conventions: https://kotlinlang.org/docs/coding-conventions.html
- Idioms: https://kotlinlang.org/docs/idioms.html

用于确认命名、文件组织、格式、不可变优先、常见惯用写法。

## 类型系统

- Null safety: https://kotlinlang.org/docs/null-safety.html
- Data classes: https://kotlinlang.org/docs/data-classes.html
- Sealed classes and interfaces: https://kotlinlang.org/docs/sealed-classes.html
- Inline value classes: https://kotlinlang.org/docs/inline-classes.html

用于确认 nullable 边界、数据载体、受限状态层次和单值领域类型。

## 表达式与函数

- Scope functions: https://kotlinlang.org/docs/scope-functions.html
- Extensions: https://kotlinlang.org/docs/extensions.html
- Lambdas and higher-order functions: https://kotlinlang.org/docs/lambdas.html
- Collections overview: https://kotlinlang.org/docs/collections-overview.html

用于确认 scope function 选择、扩展函数边界、集合变换和高阶函数可读性。

## 协程

- Coroutines guide: https://kotlinlang.org/docs/coroutines-guide.html
- Cancellation and timeouts: https://kotlinlang.org/docs/cancellation-and-timeouts.html
- Coroutine exceptions handling: https://kotlinlang.org/docs/exception-handling.html
